使い方

```text
[java]
12.0.1 (JDKバージョンエラーに注意)

[実行手順]

git clone https://github.com/takkii/Akisui.git
cd ~/Akisui/out/artifacts/Akisui_jar
java -jar Akisui.jar

[ フォルダ/ファイル 指定 ]
(例) D:/Akisui/src/Akisui.java

[ 探したい文字列 or 正規表現 ]
$ System
or
$ [S]

※ Ruby製zinbeijettのengineより強力で私のこだわりに近いです。
```
